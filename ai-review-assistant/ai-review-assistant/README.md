# AI Code Review Assistant

A Spring Boot REST API that integrates with a locally running Ollama model (mistral) to review source code.

## Tech Stack

- Java 21 (Temurin)
- Spring Boot 3.2.5
- Spring WebFlux (Netty)
- Maven
